jQuery

jQuery 是一個快速、小巧且功能豐富的 JavaScript 函式庫，使 HTML 文檔遍歷和操作、事件處理、動畫以及 AJAX 的簡化變得更加輕鬆。

使用示範

你已經點擊了按鈕！

代碼示範

            
$(document).ready(function(){
    $("#test-button").click(function(){
        $("#test-text").toggle();
    });
});

安裝方法

使用 CDN 載入 jQuery：

            
<script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>

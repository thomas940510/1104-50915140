Lottie 動畫

通過 Lottie.js 展示高品質的 JSON 動畫，適合增強網頁的視覺效果。

使用示範

代碼示範

            
const animation = lottie.loadAnimation({
    container: document.getElementById('lottie-animation'),
    renderer: 'svg',
    loop: true,
    autoplay: true,
    path: 'https://lottie.host/dcefc271-a13f-4e04-9265-69287723360f/5HbeWaBNSg.json' // JSON 動畫檔案路徑
});
